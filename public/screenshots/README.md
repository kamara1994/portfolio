Each project card has a screenshot area. Add images here matching these filenames:

| File | Project |

### Requirements:
- Format: PNG or JPG
- Recommended size: 1200x675px (16:9)
- Keep under 400KB each (use tinypng.com to compress)

### What makes a good project screenshot:
- Dashboard views showing live data
- Terminal output showing the tool working
- Architecture diagrams
- n8n workflow screenshots
- Splunk dashboard views

### If you don't have screenshots yet:
The cards show a placeholder with a helpful message. Add screenshots as you build them.
