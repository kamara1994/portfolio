# 📁 /public/profile/

## Add Your Profile Photo Here

Drop your photo in this folder and name it:

```
joseph.jpg
```

That's it. The site will automatically display it in the Hero section.

### Requirements:
- File name: `joseph.jpg` (or update the path in `components/Hero.tsx`)
- Recommended size: 400x400px minimum (square)
- Format: JPG or PNG
- Keep file size under 500KB for fast loading

### If you want to use a PNG instead:
Open `components/Hero.tsx` and change:
```tsx
src="/profile/joseph.jpg"
```
to:
```tsx
src="/profile/joseph.png"
```
